# 🩺 Smart Pregnancy Care System

## 📌 Problem Statement
Pregnant women lack continuous, personalized monitoring of their physical and mental health, leading to undetected risks and complications for both mother and baby.

## 💡 Solution
A smart digital system that tracks pregnancy, monitors health, predicts risks, and provides personalized guidance to ensure a safe and healthy journey for mother and baby.

## 🚀 Features
- Pregnancy stage tracking (embryo to baby growth)
- Disease risk prediction (heart, diabetes, etc.)
- Mental health mood tracker
- Personalized health tips
- Alerts & recommendations

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python / Node.js
- Database: MongoDB
